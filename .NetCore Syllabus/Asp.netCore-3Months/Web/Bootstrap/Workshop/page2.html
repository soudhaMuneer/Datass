Understanding the Grid System

Bootstrap's grid system is based on a 12-column layout. You can create rows and divide them into columns using Bootstrap's predefined classes.

Add a new div inside the body element to create a container:

<div class="container">
  <!-- Your content goes here -->
</div>

Inside the container, add a new div for the row:

<div class="container">
  <div class="row">
    <!-- Your columns go here -->
  </div>
</div>

Inside the row, add three div elements for columns:

<div class="container">
  <div class="row">
    <div class="col-sm">Column 1</div>
    <div class="col-sm">Column 2</div>
    <div class="col-sm">Column 3</div>
  </div>
</div>
