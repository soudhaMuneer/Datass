Setting up the Environment


Follow these steps to set up the environment for your Bootstrap project:

  • Create a new folder for your project.
  • Inside the folder, create a new file called index.html.
  • Open the index.html file in a text editor.
  • Add the following basic HTML structure:
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Bootstrap Grid System Workshop</title>
  <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">
</head>
<body>
  <!-- Your content goes here -->

  <script src="https://code.jquery.com/jquery-3.5.1.slim.min.js"></script>
  <script src="https://cdn.jsdelivr.net/npm/@popperjs/core@2.5.4/dist/umd/popper.min.js"></script>
  <script src="https://stackpath.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>
</body>
</html>

This code includes the necessary HTML structure and links to the Bootstrap CSS and JavaScript files from a CDN (Content Delivery Network).
