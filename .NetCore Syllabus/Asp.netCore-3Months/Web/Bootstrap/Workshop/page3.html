Experimenting with Different Grid Classes

Change the column classes to create different layouts:

Rows and Columns

In the Bootstrap grid system, you start by creating a row using the class .row. Inside the row, you can add columns using classes like .col-md-4, .col-sm-6, and so on. The number after col- represents the screen size (md for medium, sm for small, etc.), and the number after the hyphen represents the number of columns the element should span (out of the 12 available columns).

Responsive Grid Classes

Bootstrap provides responsive grid classes that allow you to specify the width of a column based on the screen size. Here are some common classes:

  • .col-xs-* (extra-small screens, like mobile phones)
  • .col-sm-* (small screens, like tablets)
  • .col-md-* (medium screens, like laptops)
  • .col-lg-* (large screens, like desktops)
  • .col-xl-* (extra-large screens, like large desktops)

The * represents a value between 1 and 12, which specifies the number of columns the element should span.
