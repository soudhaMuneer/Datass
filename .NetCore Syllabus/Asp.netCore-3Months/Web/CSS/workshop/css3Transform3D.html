CSS3 3D-TRANSFORM

1.RotateX method used to rotate the element around its X-axis at a given degree
2.RotateY method used to rotate the element around its Y-axis at a given degree.
3.Scale method used to increases or decreases the size of the element depending on the parameters given for the width and the height.
4.Skew method used to turns the element in a given angle depending on the parameters given for the horizontal and the vertical lines.
5.The matrix method combines all of the 2D transform methods into one.
