2.to create columns of text, similar to how you might see in a newspaper.

column-count: Specifies the number of columns.

column-gap: Specifies the gap between columns.

column-width: Specifies the ideal width of each column. The browser will adjust the number of columns to accommodate this width.

column-rule: Adds a rule (line) between columns.
